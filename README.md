# 去哪儿网APP

> 利用周末时间用vue全家桶制作去哪儿网APP

## 技术栈
- vue2
- vue-router
- webpack
- axios
- vuex
- stylus
- es6
- vue-awesome-swiper
- iconfont

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
